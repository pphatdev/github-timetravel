**⌚ Make your GitHub history back to 2001.**

## Travel Back ⌚

### How ?
1. Create repository from [github.com](https://github.com/) named `2001`
2. [Generate a personal access token](https://github.com/settings/tokens/new) on GitHub and copy it.
3. Goto open `gitbash` Run the following script:

```bash
$ sh -c "$(curl -fsSL https://raw.githubusercontent.com/pphatdev/github-old/master/index.sh)"
```
